Content Management

Gestión de contenidos: Hay muchas maneras de administrar el contenido de manera eficiente.
 Ya sea un sitio web de CMS, publicación multicanal o comercio de contenido.

Xiib puede ayudar con soluciones a medida para una gestión de contenido eficiente.


La gestión de contenidos significa almacenar, gestionar y procesar datos y documentos. 
La gestión de contenidos vive en la interfaz entre la comunicación, la tecnología, la organización y el diseño. 


El contenido es muy versátil: el desafío es administrar y publicar este contenido de manera estructurada y eficiente. 
Debido a esta versatilidad, se debe tomar una decisión individual para cada aplicación en cuanto a qué solución es adecuada.

Sistemas de gestión de contenidos individuales.

Desarrollamos sistemas de gestión de contenidos a medida. Basado principalmente en tecnología de código abierto . Los casos de uso son:

  • Sites
  • Web de campañas multilingües
  • Páginas de destino / Landing Pages

Estas soluciones son ideales para sitios web de estructura clásica que siguen una estructura de árbol de página definida y se basan en tipos de página.

Headless content management systems. Sistemas de gestión de contenido sin cabeza

storyblok y contentful son sistemas CMS que comúnmente se anuncian bajo el término "CMS sin encabezado" (headless CMS) o .o "plataforma de contenido" (content platform)

Los sistemas de administración de contenido individual son ideales para proporcionar contenido para un caso de uso específico. Las plataformas de contenido como storyblok y contentful son ideales para definir el contenido de forma centralizada y hacer que este contenido esté disponible para varias aplicaciones, canales y puntos de contacto digitales.

El contenido se divide en los llamados bloques de contenido (contenido estructurado) y estos bloques están disponibles para una amplia variedad de aplicaciones a través de una API.

Ejemplos de bloques de contenido estructurado:

  • Textos legalmente vinculantes como la impresión y la protección de datos
  • Componentes digitales (banners publicitarios, etc.)
  • Información del producto, como la designación del producto, precios, campañas de descuento por tiempo limitado
  • Horarios e información de ubicación de puntos de venta y oficinas

Propuesta de valor:

  • Cree una vez, publique en cualquier lugar
  • Tiempo de comercialización más rápido
  • Cargas de página más rápidas

Los casos de uso son:

  • Witios web corporativos
  • Sitios de comercio electrónico
  • Aplicaciones móviles
  • señalización digital
