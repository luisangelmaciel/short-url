GitHub Redirection

The JavaScript of this CodePen creates a modal that redirects users to GitHub. Click on the "Show Example" button below to see an example of the modal that is created.

It can be added to any project by simply linking to this CodePen's JavaScript file.

https://codepen.io/mca62511/pen/qZoPzM.js

After being added to a project, the modal can be generated in the following way.

gitHubRedirect.show( "https://github.com/username/project" );

If you want to supply a custom title or message, then pass an object.

gitHubRedirect.show({
    url: "https://example.com",
    title: "Message Title!",
    message: "Your message goes here."
});

Click on the button bellow to see the modal that this code would produce.

Additionally, if the version of your project is no longer working on CodePen, you can add a canContinue key with a value of true to remove the "Continue on CodePen" button. This will also update the default message to indicate that the CodePen version of the project is no longer working.
