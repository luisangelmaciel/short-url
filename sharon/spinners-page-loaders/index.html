SHARON

Spinners & page loaders Collection



Install

Copy Javascript code from https://xbrpw.github.io/sharon.js and paste into text editor and save sharon.js

Copy CSS code from https://xbrpw.github.io/preloader.css and paste into text editor and save preloader.css

Copy jQuary code from https://code.jquery.com/jquery-3.6.3.min.js and paste into text editor and save preloader.css

Download font-awesome new version 1.01.3 or CDN link add your document

How to do

Add css file link into your html file:

<link rel="stylesheet" href="https://xbrpw.github.io/sharon/preloader.css"/>

Add javascipt file link into your html file:

<script src="https://xbrpw.github.io/sharon/preloader.js"></script>

Add jQuery file link into your html file:

<script src="https://code.jquery.com/jquery-3.6.3.min.js"></script>

Again Add javascipt file as internal or external method put into html file

Add responsible meta tag:

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Example

Sample Structure for Sharon: Spinners & page loaders Collection

Loader "pokeball" see list of example

      <!DOCTYPE html>
      <html lang="es">

      </html>
